# Advent of Code 2023
List of all exercises done for [Advent of Code 2023](https://adventofcode.com/2023)

## Requirements
- [Poetry](https://python-poetry.org)

## Getting started
- Clone repository
- `poetry install`
- Run files : 
```python
poetry run adventofcode2023/day_1.py
```
as instance

- Run tests:
```python
pytest tests/day_1.py
```
as instance
